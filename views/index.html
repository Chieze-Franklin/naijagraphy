Features

Send simple GET requests to intuitive URLs to get any info you want.

To get all the states in Nigeria:

Send a GET request to
https://naijagraphy.herokuapp.com/states.
This will return a JSON array containing the names of all the states in Nigeria, including the FCT Abuja.

To get basic info about a particular state:

Send a GET request to
https://naijagraphy.herokuapp.com/states/{state}.
This will return a JSON object containing basic info (like name and capital) about the specified state.
For instance,
naijagraphy.herokuapp.com/states/Lagos.

To get more info about a particular state:

Send a GET request to
https://naijagraphy.herokuapp.com/states/{state}/{info}.
For instance,
naijagraphy.herokuapp.com/states/Lagos/lgas
This will return a JSON array containing all the LGAs in Lagos state.

Contribute

There is no "hard-coded" list of states, capitals or local government areas in this solution; instead the various data are fetched from this Github repository.
So if, in addition to LGAs, you also want data about population of a state to be returned when a request is made to a URL like
naijagraphy.herokuapp.com/states/Lagos/population
head over to the repository, and make a pull request.
